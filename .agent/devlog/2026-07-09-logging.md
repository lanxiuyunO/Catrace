# 2026-07-09 统一日志系统

## 会话目标

解决用户反馈排查困难的问题：整个 exe 几乎没有日志，前端也无法查看后端日志。

## 完成项

- release 构建启用 devtools，右键菜单可打开控制台
- 后端统一日志写入本地文件，按天轮转、保留 7 天
- Debug 页增加「日志目录」按钮，用户可直接打包日志反馈
- 前端 `console.log/warn/error` 自动转发到后端，写入同一日志文件
- 移除之前丑陋且不实用的全局浮窗 LogViewer

## 关键文件变更

| 文件 | 变更 |
|------|------|
| `src-tauri/Cargo.toml` | tauri 启用 `devtools` feature |

| `src/i18n/locales/zh-CN.ts` / `en-US.ts` | 新增 `debug.openLogsDir` |
